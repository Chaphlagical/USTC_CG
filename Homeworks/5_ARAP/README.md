## 作业要求

- 在给定的网格框架上完成作业，实现
  - ASAP (As-similar-as-possible) 参数化算法
  - ARAP (As-rigid-as-possible) 参数化算法

- 对各种参数化方法（包括作业4的Floater方法、ASAP/ARAP方法等）进行比较

- 学习和实现矩阵的 SVD 分解
- 进一步巩固使用 Eigen 库求解大型稀疏线性方程组
